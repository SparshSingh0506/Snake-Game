# Snake Game (C++ + Raylib)

Classic Snake game made using **C++** and the **Raylib** graphics library. 
This project was built to practice and deepen my knowledge of OOP concepts.

---

## Tech Stack

- **Language**: C++
- **Graphics Library**: Raylib
- **Build**: g++ / Makefile

---

## How to Run

- Download the zip file
- Run "Snake.exe"
